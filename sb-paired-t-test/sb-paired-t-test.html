What kinds of Variables work with a Paired t Test

  • One scalar (a.k.a. “measurement”) variable
  • One nominal variable with only two categories or two categories that you have selected.

Background

For a thorough treatment of the background for t tests generally, see our lessons One-Sample t Test and Two-Sample t Test.

Use the paired t–test when you have one measurement variable and two nominal variables, one of the nominal variables has only two values, and you have only one observation for each combination of the nominal variables; in other words, you have pairs of observations rather than unrelated observations.

The most common design is that one nominal variable represents individual organisms, while the other is “before” and “after” some treatment. Sometimes the pairs are spatial rather than temporal, such as left vs. right, injured limb vs. uninjured limb, etc. You can use the paired t–test for other pairs of observations; for example, you might sample an ecological measurement variable above and below a source of pollution in several streams.

The paired t test shows whether the mean difference in the pairs is different from 0.

Paired t-tests can be conducted with the t.test function in the native stats package using the paired = TRUE option. Data can be in long format or short format.

Null and Alternative Hypotheses

  • H0: "The mean difference between paired observations is zero.

When the mean difference is zero, the means of the two groups must also be equal. Because of the paired design of the data, the null hypothesis of a paired t–test is usually expressed in terms of the mean difference.

  • HA: “The mean difference between paired observations is not zero.”
  • HA (1-sided): “The mean difference between paired observations is a positive number.”
  • HA (1-sided): “The mean difference between paired observations is a negative number.”

Assumptions

The paired t–test assumes that the differences between pairs are normally distributed; you can use a histogram to check the normality. If the differences between pairs are severely non-normal, it would be better to use the Wilcoxon signed-rank test.

We don’t think the paired t test is very sensitive to deviations from normality, so unless the deviation from normality is really obvious, you shouldn’t worry about it.

The paired t–test does not assume that observations within each group are normal, only that the differences are normal. And it does not assume that the groups are homoscedastic.

How to run the Test

As mentioned above, you can do the test with wide or long data.

Paired t-test, data in wide format, flicker feather example

I loaded a data set for you for this example called birds. Use head(birds) to see what the top of the data file looks like.

head(birds)

Pass the two variables to t.test, set paired equal to TRUE, and set conf.level to 0.95 as usual.

t.test(birds$Typical, 
       birds$Odd, 
       paired=TRUE, 
       conf.level=0.95)
